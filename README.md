## Features :

* Users can add notes 
* Pinned and unpinned notes
* Edit a note
* Delete a note
* Add colors to note
* Archive a note
* Unarchive a note
* Move notes to trash
* Sort notes by label, time created
* Search Notes
* Authentication - Login, Signup, Logout
* Responsive
* Dark and light mode

## Technologies Used :

* ReactJS
* React Context API + useReducer
* React Router v6
